# AFO 왕국 헬스 게이트 규칙 (SSOT - 세종 모드 정의)
# 2026-01-06 세종 모드에서 게이트 고정

# 판정 규칙 (항상 이 라인이 헬스체크 출력에 포함되어야 함)
OVERALL=GREEN WARN=model_switching(memory)

# 게이트 정책 (A안 선택: 운영 평온 최우선)
# - GREEN: 핵심 운영 정상 (connectivity, fallback)
# - WARN: 기능 일부 제한 (model_switching 메모리 이슈)
# - RED: 시스템 다운 (연결 실패 등 치명적 문제)

# CI/게이트 적용 규칙
# - WARN 허용: 배포/운영 계속 진행
# - RED만 FAIL: 진짜 문제만 차단
# - GREEN 최우선: 안정성 유지

# 재발 방지 (영속성)
# - 이 규칙은 헬스체크 코드에 하드코딩되어야 함
# - 변경 시 세종 모드 승인 필수
# - 판정 논쟁 시 이 파일이 SSOT 증거
