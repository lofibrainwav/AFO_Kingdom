## SSOT 원칙 요약

1. **과장 표현 제거**: "100%", "완벽", "모든" 등은 측정 가능한 근거가 있어야 함
2. **할루시네이션 금지**: 실제로 존재하는 파일/자료만 언급
3. **외부 인물 언급**: 실제 링크/출처 제공 또는 언급 제거
4. **평가 근거**: "100/100" 등은 측정 가능한 기준과 근거 필요
5. **비존재 자료 언급 금지**: 첨부되지 않은 이미지/PDF는 언급하지 않음
