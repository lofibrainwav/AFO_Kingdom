## 🎨 Mermaid 다이어그램 목록

### 1. 시스템 아키텍처
- 전체 시스템 구조를 한눈에 볼 수 있는 다이어그램
- 사령관 → 승상 → MCP 서버 → Skills/Context7 흐름

### 2. Antigravity & Chancellor 통합 흐름
- 시퀀스 다이어그램으로 통합 과정 시각화
- 사용자 요청부터 응답까지의 전체 흐름

### 3. Skills Registry 구조
- 19개 스킬의 구조와 MCP 변환 과정
- 주요 스킬 하이라이트

### 4. Trinity Score 시스템
- 5기둥 (眞善美孝永) 계산 과정
- 가중치와 최종 점수 산출 흐름

### 5. 데이터 흐름
- 전체 시스템의 데이터 흐름도
- 각 컴포넌트 간 상호작용 시각화

## 💡 옵시디언 사용 팁

### 기본 기능

1. **Mermaid 렌더링**
   - 옵시디언은 Mermaid를 기본 지원
   - 플러그인 없이 자동으로 렌더링됨
   - 다이어그램을 클릭하면 확대/축소 가능

2. **다이어그램 편집**
   - 소스 모드에서 Mermaid 코드 직접 수정 가능
   - 변경 사항이 실시간으로 반영됨

3. **전체 화면 보기**
   - 다이어그램을 클릭하면 전체 화면으로 확대
   - ESC 키로 원래 화면으로 복귀

### 추천 플러그인 (선택사항)

1. **Mermaid Tools**
   - Mermaid 다이어그램 편집 기능 강화
   - 설치: 설정 → 커뮤니티 플러그인 → "Mermaid Tools" 검색

2. **Advanced Tables**
   - 표 편집 기능 강화
   - 설치: 설정 → 커뮤니티 플러그인 → "Advanced Tables" 검색

3. **Kanban**
   - 칸반 보드 지원
   - 설치: 설정 → 커뮤니티 플러그인 → "Kanban" 검색
