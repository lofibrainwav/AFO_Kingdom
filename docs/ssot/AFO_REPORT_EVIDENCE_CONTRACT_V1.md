# AFO_REPORT_EVIDENCE_CONTRACT_V1 (SSOT)

## Strict 6-line Evidence Block
Exactly 6 lines, in this order, each line starts with the prefix:

E1_REF: Time & Git Context
E2_INFRA: Port Status (8010, 3000)
E3_HTTP: HTTP Status Codes (GET)
E4_VIEW: Dashboard Title
E5_ENV: Runtime Environment (Playwright)
E6_SEAL: Verification Script Status (ssot_verify.sh)

## Rules
- Exactly 6 lines.
- No extra lines, no blank lines.
- Must be generated by: scripts/afo_report_evidence_v1.sh
- Script should finish < 5 seconds even if services are down.
- Reports are append-only: do not hand-edit evidence lines.

## Example
E1_REF: time=2026-01-09T00:00:00-08:00 branch=main sha=abc1234
E2_INFRA: 8010=OPEN 3000=DOWN
E3_HTTP: 8010/health=200 3000/=000
E4_VIEW: title="Trinity Dashboard - AFO Kingdom"
E5_ENV: playwright=OK version=1.40.0
E6_SEAL: ssot_verify.sh=TIMEOUT exit=124
