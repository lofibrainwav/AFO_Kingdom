## 📊 평가 기준

### 정확성 (Accuracy)
- **○ (Excellent)**: IRS 공식 계산과 100% 일치
- **△ (Good)**: 소소한 차이 있지만 사용 가능
- **× (Poor)**: 큰 오류나 계산 실패

### 사용성 (Usability)
- **5점**: 매우 직관적, 교육 불필요
- **4점**: 약간의 학습 필요하지만 편리
- **3점**: 사용 가능하지만 개선 필요
- **2점**: 불편함, 자주 헤맴
- **1점**: 매우 어렵거나 불가능

### 신뢰성 (Reliability)
- **5점**: 완전히 신뢰, Evidence Bundle 만족
- **4점**: 대체로 신뢰할 만함
- **3점**: 약간의 의구심
- **2점**: 신뢰하기 어려움
- **1점**: 전혀 신뢰할 수 없음
