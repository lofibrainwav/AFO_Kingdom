# AFO Kingdom 오픈소스 전략

## 세종대왕 정신 (世宗大王 精神)

> "기존 오픈소스를 찾아 → 왕국 커스텀으로 확장·고쳐쓰기"

### 핵심 원칙

| 원칙 | 세종대왕 | AFO Kingdom |
|------|----------|-------------|
| **실용 흡수** | 한자·중국 기술 활용 | 기존 OSS 포크 |
| **왕국 커스텀** | 훈민정음·측우기 창제 | 眞善美孝永 철학 주입 |
| **문화 주권** | 조선 독자 문자·과학 | MIT/Apache 선택 |
| **기여 환원** | 백성 교육·보급 | Upstream PR |

## 라이선스 우선순위

1. **MIT** (최우선) - 최대 자유
2. **Apache 2.0** - 특허 보호
3. **BSD** - MIT와 유사

### ⚠️ 피해야 할 라이선스

- **AGPL** - SaaS 모델 위험
- **GPL** - 상업 제한
