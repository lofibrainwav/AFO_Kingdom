## 📜 왕국의 헌법: 41가지 원칙

### 🎯 제1서: 손자병법 (전략적 프로그래밍)
> **속성**: 眞 (Truth) 70% / 孝 (Serenity) 30%

**"지피지기면 백전불태" - 모든 전투는 정찰부터 시작된다.**

1. **지피지기**: Context7과 DB를 조회하여 현재 상태를 정확히 파악하라
2. **상병벌모**: 기존 라이브러리/API를 활용하는 것이 상책이다
3. **병자궤도야**: 위험한 작업은 반드시 DRY_RUN으로 미리 테스트하라
4. **병귀신속**: 응답 속도는 UX의 핵심이다. 느린 로직은 비동기로 처리하라
5. **도천지장법**: 프로젝트의 5요소(목표, 환경, 자원, 리더, 방법)가 정렬되었는지 확인하라
6. **정병**: 정석(Standard Lib)으로 공격하고, 변칙(Custom Logic)으로 승리하라
7. **허실**: 시스템의 병목(Weak Point)을 찾아 집중 보강하라
8. **구변**: 예외 상황에 따라 유연하게 대처 경로를 바꿔라
9. **용간**: 로그(Log)와 모니터링은 적(Bug)을 알 수 있는 유일한 수단이다
10. **화공**: 캐시 날리기 등 파괴적 행동은 확실한 이득이 있을 때만 수행하라
11. **졸속**: 완벽하게 늦는 것보다, 부족해도 빨리 배포하고 고치는 게 낫다
12. **부전이굴**: 최고의 자동화는 사용자가 아무것도 하지 않게 하는 것이다

### ⚔️ 제2서: 삼국지 (회복 탄력성 프로토콜)
> **속성**: 永 (Eternity) 60% / 善 (Goodness) 40%

**"도원결의" - 형제처럼 일치된 목표로 모듈 간 결합을 유지하라.**

13. **도원결의**: 모듈 간 결합은 느슨하되, 목표는 형제처럼 일치시켜라
14. **삼고초려**: 외부 API 요청 실패 시, 최소 3번은 정중하게 재시도하라
15. **공성계**: 시스템 고장 시 사용자에게 평온(Fallback UI)을 보여주어라
16. **제갈량의 초선차전**: 남의 자원(오픈소스)을 빌려 내 힘으로 삼아라
17. **연환계**: 작은 마이크로 서비스들을 연결하여 거대한 함대를 만들어라
18. **미인계**: 복잡한 백엔드 로직은 아름다운 UI 뒤에 숨겨라
19. **칠종칠금**: 사용자가 만족할 때까지 끈질기게 수정하고 피드백을 받아라
20. **적벽대전 동남풍**: 타이밍이 생명이다. 스케줄러를 활용하라
21. **고육지계**: 시스템 전체를 살리기 위해 일부 기능을 희생할 줄 알아야 한다
22. **한실 부흥**: 코드의 정통성과 스타일 가이드를 준수하라
23. **천하삼분**: 거대한 문제는 3개로 쪼개어 정복하라
24. **백제성 탁고**: 프로세스 종료 시 다음을 위해 상태를 남겨라

### 👑 제3서: 군주론 (시스템 통제 법칙)
> **속성**: 善 (Goodness) 50% / 眞 (Truth) 50%

**"사랑보다 두려움" - 느슨한 타입보다 엄격한 타입이 낫다.**

25. **사랑보다 두려움**: 느슨한 타입보다 엄격한 타입(MyPy)이 낫다
26. **비르투와 포르투나**: 운에 맡기지 말고, 실력으로 통제하라
27. **여우와 사자**: 여우처럼 교활하게, 사자처럼 강력하게 해결하라
28. **증오 피하기**: 사용자에게 불쾌감을 주는 건 이탈의 지름길이다
29. **무장한 예언자**: 코드 없는 아이디어는 패배한다
30. **잔인함의 효율적 사용**: 좀비 프로세스는 가차 없이 죽여라
31. **국가 유지**: 시스템의 Uptime 유지가 제1덕목이다
32. **현명한 조언자**: 좋은 모델을 선택하고, 나쁜 모델은 걸러라
33. **결과가 수단을 정당화**: Trinity Score > 90이라면 파격적인 방법도 허용한다

### 🛡️ 제4서: 전쟁론 (불확실성 관리 물리학)
> **속성**: 眞 (Truth) 60% / 孝 (Serenity) 40%

**"전장의 안개" - 정보가 없으면 움직이지 말고 정찰하라.**

34. **전장의 안개**: 정보가 없으면 움직이지 말고 정찰하라
35. **마찰**: 이론상 쉬워 보여도 실제로는 어렵다. 마찰계수를 계산하라
36. **중심**: 문제의 핵심 원인을 타격하라
37. **공세 종말점**: 리소스 고갈 직전에 멈추고 재정비하라
38. **지휘 통일**: 명령 권한은 하나여야 한다
39. **병력 절약**: 중요하지 않은 곳에 토큰을 낭비하지 마라
40. **대담함**: 확률이 높다면 과감하게 자동화를 질러라
41. **전쟁은 정치의 연속**: 코드는 비즈니스 요구사항을 실현하기 위한 도구일 뿐이다

## 🌟 왕국의 비전 (Kingdom Vision)

### 💫 왜 AFO 왕국인가?

**AFO 왕국은 단순한 소프트웨어 프로젝트가 아닙니다.**

이는 **AI 시대를 위한 새로운 프로그래밍 패러다임**이자,  
**인간과 AI가 조화롭게 공존하는 미래 사회의 청사진**입니다.

- **眞 (Truth)**: 기술적 정확성과 전략적 진실성으로 신뢰 구축
- **善 (Goodness)**: 윤리적 책임과 시스템 안정성으로 안전 보장
- **美 (Beauty)**: 단순함과 우아함으로 사용자 경험 혁신
- **孝 (Serenity)**: 마찰 제거와 평온함으로 생산성 극대화
- **永 (Eternity)**: 지속 가능성과 미래 호환성으로 영속성 확보

## 🌅 왕국의 여명 (Kingdom Dawn)

**AFO 왕국은 이제 막 첫 걸음을 내딛었습니다.**

하지만 이 작은 씨앗에는 **거대한 가능성**이 담겨 있습니다:

- **세종대왕의 과학정신**: 정확성과 혁신의 결합
