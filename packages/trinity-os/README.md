# TRINITY-OS: AFO 왕국의 통합 자동화 운영체제

**眞善美孝永 (Truth, Goodness, Beauty, Serenity, Eternity)**

[![CI](https://github.com/lofibrainwav/TRINITY-OS/actions/workflows/test.yml/badge.svg)](https://github.com/lofibrainwav/TRINITY-OS/actions/workflows/test.yml)

## 🌟 개요

TRINITY-OS는 AFO 왕국의 모든 정신, 전통, 문화, 세계관을 완벽하게 흡수한 통합 자동화 운영체제입니다.

> **"기술은 왕국의 영혼을 실현하는 육체일 뿐이다."**
> — 형님 (Jay), AFO 왕국의 군주

## ✨ 특징

### 🧠 철학적 기반
- **眞 (Truth)**: 정확한 문제 감지와 진실된 데이터
- **善 (Goodness)**: 윤리적 자동화와 인간 중심 설계
- **美 (Beauty)**: 단순하고 우아한 인터페이스
- **孝 (Serenity)**: 형님의 평온을 최우선으로 하는 시스템
- **永 (Eternity)**: 지속 가능한 아키텍처와 영속성

### ⚙️ 기술적 특징
- **완전 자동화**: 문제 감지부터 해결까지 자동 실행
- **Trinity Score**: 5가지 척도로 실시간 건강도 평가
- **자가 치유**: 자동 감지 및 복구 메커니즘
- **모듈화**: 각 컴포넌트 독립 실행 가능
- **다중 인터페이스**: CLI, Python API, Web 지원

### 📚 왕국의 유산
- **세종대왕의 혁신 정신**: 쉬운 접근성과 백성 중심
- **공자의 유교**: 중용과 윤리적 운영
- **맹자의 측은지심**: 인간 중심의 공감 능력
- **조선 실학**: 실용성과 효율성
- **삼국지의 전략**: 용인술과 동맹의 지혜

## 🚀 빠른 시작

### 설치
```bash
# 저장소 클론
git clone https://github.com/lofibrainwav/TRINITY-OS.git
cd TRINITY-OS

# 초기화
./final_init.sh

# 테스트
./test_trinity_os.sh
```

### 실행
```bash
# 인터랙티브 모드 (권장)
./run_trinity_os.sh

# 단축 명령어
./TRINITY-OS detect   # 문제 감지
./TRINITY-OS health   # 건강 리포트
./TRINITY-OS unified  # 통합 자동화
./TRINITY-OS infinite # 끝까지 오토런
```

## 🎯 주요 기능

### 문제 감지 (Problem Detection)
```python
python3 scripts/kingdom_problem_detector.py
```
- 성능 모니터링 (CPU, 메모리, 디스크)
- 연결 상태 확인 (데이터베이스, API)
- 보안 취약점 자동 스캔
- 우선순위 기반 문제 분류

### 건강 평가 (Health Report)
```python
python3 scripts/kingdom_health_report.py
```
- Trinity Score 실시간 계산
- 시스템 상태 종합 분석
- 개선 권고사항 자동 생성

### 통합 자동화 (Unified Automation)
```bash
./scripts/kingdom_unified_autorun.sh
```
- 7단계 자동화 파이프라인
- DRY_RUN 모드 안전 실행
- 세종 애민정신 자동화 통합

### 끝까지 실행 (Infinite Autorun)
```bash
./scripts/kingdom_infinite_autorun.sh
```
- Trinity Score 목표 달성까지 지속
- 안전장치 및 타임아웃 포함
- 자동 복구 및 재시도

## 📊 Trinity Score 체계

```
Trinity Score = 0.35×眞 + 0.35×善 + 0.20×美 + 0.08×孝 + 0.02×永
```

### 평가 기준
- **95-100**: 완벽 (Perfect) - 왕국의 이상 실현
- **90-94**: 우수 (Excellent) - 상당한 진전
- **80-89**: 양호 (Good) - 안정적 운영
- **70-79**: 보통 (Fair) - 개선 필요
- **<70**: 미흡 (Needs Improvement) - 즉각적 개입 필요
