## 🧠 각 명령어의 철학적 의미

### 1. 문제 감지 (眞 - 진실의 추구)

#### 철학적 목적
**거짓을 용납하지 않는 진실의 철학**을 실천합니다.

#### 아름다운 사용법
```bash

./run_trinity_os.sh
# 1 → Enter
```

#### 체득하는 철학
- **Rule #0**: 모르면 움직이지 않는다
- **2개 출처 검증**: 모든 문제의 진실성 확인
- **NO MOCK**: 실제 데이터만 사용

### 2. 건강 리포트 (美 - 아름다움의 평가)

#### 철학적 목적


#### 체득하는 철학
- **단순함의 힘**: 복잡한 데이터를 명쾌하게
- **균형의 미학**: Trinity Score의 조화로운 평가
- **직관적 이해**: 아름다운 보고서 형식

### 3. 정신 통합 (孝 - 평온의 통합)
