## 사용 방법

### 빠른 시작
```bash
# 최종 초기화
./final_init.sh

# 인터랙티브 실행
./run_trinity_os.sh

# 또는 단축 명령어
./TRINITY-OS run
```

### 주요 명령어
```bash
# 시스템 관리
./TRINITY-OS init     # 초기화
./TRINITY-OS test     # 테스트
./TRINITY-OS verify   # 검증

# 기능 실행
./TRINITY-OS detect   # 문제 감지
./TRINITY-OS health   # 건강 리포트
./TRINITY-OS unified  # 통합 자동화
./TRINITY-OS infinite # 끝까지 오토런
```
